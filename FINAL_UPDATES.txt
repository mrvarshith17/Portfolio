CUSTOMIZATION
==============

Want to adjust animations?

1. CHANGE SPEED
   Edit tailwind.config.ts
   Change duration values (0.6s, 0.8s, 1s, etc.)

2. DISABLE ANIMATIONS
   Remove animate-* class from section
   <section className=""> instead of <section className="animate-clip-reveal-left">

3. MODIFY COLORS
   Edit Header.tsx coding platform colors
   Change Orange → Blue, etc.

4. ADD MORE PROFILES
   Copy coding platform section in Header.tsx
   Update URL and styling
