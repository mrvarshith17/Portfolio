WHAT HAPPENS WHEN YOU NAVIGATE:
═══════════════════════════════════════════════════════════════════════════════

Example: You're on Projects and click "Responsibilities"

CURRENT STATE:
  ├─ You're viewing Projects section (scrolled to bottom)
  └─ activeSection = "projects"

CLICK BUTTON:
  ├─ Event fires: scrollToSection("position-of-responsibility")
  ├─ exitingSection = "projects" ← Projects starts fading out
  └─ New animation type = "clip-reveal-bottom"

DURING TRANSITION (0-300ms):
  ├─ Projects section opacity decreases
  ├─ Projects section still visible but fading
  └─ User sees smooth fade-out effect

NAVIGATION BEGINS (300ms):
  ├─ activeSection = "position-of-responsibility"
  ├─ Page smoothly scrolls to Responsibilities section
  ├─ Responsibilities section enters viewport
  └─ Entrance animation starts

ENTRANCE ANIMATION (300-1100ms):
  ├─ clip-reveal-bottom plays (0.8s)
  ├─ Responsibilities slides in from bottom
  └─ User sees professional section reveal

ANIMATION COMPLETE (1100ms):
  ├─ All animations done
  ├─ Responsibilities fully visible
  └─ Ready for next navigation
